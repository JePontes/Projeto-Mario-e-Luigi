<h1>🛠️ Encanadores Mario & Luigi 🛠️</h1>
<h2>Esse foi um projeto que eu aprendi no <a href="https://www.rodolfomori.com/">DevClub</a>
<h3>Projeto desenvolvido utilizando HTML e CSS para criar uma página de apresentação de serviços de encanamento inspirada nos personagens Mario e Luigi.</h3>
<p>🚀 Sobre o projeto: <br/>Este projeto consiste em uma landing page que apresenta os serviços de encanadores fictícios Mario e Luigi. <br/>A página possui layout moderno, vídeo de fundo, botão de contato e formulário para solicitar orçamento.</p>
<p>🎨 Funcionalidades: <br/>- Layout moderno com HTML e CSS <br/>- Vídeo de fundo na página <br/>- Botão de contato <br/>- Formulário para solicitar orçamento <br/>- Link direto para contato via WhatsApp</p>
<p>🛠️ Tecnologias utilizadas: <br/> <img src="https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white" alt="html-logo" /> <br/> <img src="https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white" alt="css-logo" /> 
<p>🎯 Objetivo desenvolvido para praticar conceitos de HTML e CSS, como: <br/>- Estruturação de páginas <br/>- Posicionamento de elementos <br/>- Flexbox <br/>- Estilização de botões <br/>- Formulários <br/>- Vídeo como background</p>
<p>👨‍💻 Autor Projeto: <br/> Desenvolvido por Jessica Cavalcante, durante os estudos de programação.</p>
